# Screenshots

Place the following files in this directory:

| Filename | What to capture |
|---|---|
| `main-view.png` | Main window — sidebar with projects + commit tree open |
| `import-modal.png` | Import Git Pull Output dialog |
| `add-project.png` | Add Project dialog |
| `settings.png` | Settings modal |

Recommended: 1280×800 or wider, dark theme, PNG format.
